# Skill: State Management

## Propósito
Gestión de estados globales y locales en el frontend.

## Procedimiento Inicial
* Uso de Context API (ej. `CartContext`, `WishlistContext`) para estados globales.
* Uso de `useState` y `useEffect` para estados locales.
* *[Se enriquecerá con convenciones futuras sobre cuándo extraer estado local a global]*
