# Skill: Create Component

## Propósito
Guía base para la creación de nuevos componentes en React dentro del proyecto Intipa Churin.

## Procedimiento Inicial
* Mantener la estructura de componentes funcionales en `frontend/src/components/`.
* Nombrar el archivo en `PascalCase` y en inglés (ej. `NewComponent.jsx`).
* Utilizar Tailwind CSS para los estilos, priorizando diseños limpios, modernos y glassmorphism cuando aplique.
* *[Esta skill se expandirá a medida que identifiquemos patrones recurrentes en la creación de componentes]*
