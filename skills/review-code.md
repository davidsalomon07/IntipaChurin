# Skill: Review Code

## Propósito
Guía de estándares para la revisión y refactorización de código.

## Procedimiento Inicial
* Priorizar la reducción de código (DRY) sin alterar la lógica de negocio ni el estilo visual (UI/UX).
* Preservar el idioma (variables en español, nombres de archivos en inglés).
* *[Esta skill se expandirá con criterios específicos de calidad descubiertos en revisiones futuras]*
