# Skill: Debugging

## Propósito
Procedimiento estándar para la resolución de errores en el proyecto.

## Procedimiento Inicial
* Identificar si el error proviene del Backend (Node.js/Express) o Frontend (React/Vite).
* Revisar consolas y estado de la red.
* En errores críticos (ej. Stripe o Base de datos), proporcionar detalles precisos y código SQL manual en lugar de automatizar cambios de datos.
* *[Esta skill crecerá documentando los errores más comunes y sus soluciones]*
